# 技术文档

## 技术栈
- 前端框架: Vue 3
- UI组件库: Element Plus
- 状态管理: Pinia
- 路由: Vue Router
- HTTP客户端: Axios
- 构建工具: Vite

## 后端技术
- Node.js
- Express框架
- JSON Web Token (JWT)认证

## 主要组件库
- Element Plus: 提供UI组件如表格、表单、弹窗等
- Vue Router: 实现前端路由导航
- Pinia: 管理应用状态
- Axios: 处理HTTP请求

## 开发依赖
- Vite: 前端构建工具
- @vitejs/plugin-vue: Vite的Vue插件
- Sass: CSS预处理器
